# Changelog

All notable changes to the Synapse VS Code extension will be documented in this file.

## [0.1.0] - 2025-11-16

### Added
- Initial release of Synapse VS Code extension
- Syntax highlighting for all Synapse keywords, operators, and literals
- IntelliSense with keyword, function, type, and variable suggestions
- 12+ code snippets for common patterns (def, let, if, for, try, etc.)
- Real-time diagnostics for bracket matching and basic syntax errors
- Hover provider for keyword documentation
- Command palette support for:
  - `synapse.runFile` - Execute current Synapse file (Ctrl+Shift+E)
  - `synapse.runSelection` - Run selected code
  - `synapse.openREPL` - Open interactive REPL
- Output channel for viewing execution results
- Configuration options for Python path and diagnostics
- Language configuration with auto-closing pairs and bracket matching
- Full TypeScript source with strict type checking

### Documentation
- Comprehensive README with features overview and examples
- Installation and usage instructions
- Configuration guide
- Development setup instructions
- Architecture overview
